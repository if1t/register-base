# RegisterBaseWorkspace

Этот проект был создан с помощью [Angular CLI](https://github.com/angular/angular-cli) версии 18.2.21.

## Установка Node, npm и зависимостей

Установка Node:

```bash
nvm install 22.14
```

или использование текущей версии:

```bash
nvm use 22.14
```

Установка yarn классической версии:

```bash
npm install -g yarn@1.22.22
```

Установка зависимостей проекта:

```bash
yarn install
```

## Сборка библиотеки

1. Запустите `yarn build:lib` для сборки библиотеки. Артефакты сборки будут сохранены в каталоге `dist/ngx-register-base`.
2. Войти в корневой каталог библиотеки`cd projects/ngx-register-base`.
3. Запустите `yarn build` для сборки схем библиотеки (миграции). Артефакты сборки будут сохранены в каталоге `dist/ngx-register-base/schematics`

## Тестирование библиотеки или миграций

1. Сборка библиотеки.
2. В каталоге рабочего пространства свяжите локально собранный пакет через команду `npx link ./dist/ngx-register-base`. Это создаст ссылку в node_modules на библиотеку.
3. Запустить приложение.
4. Запустить отдельно миграции по интересующим версиям `ng update ngx-register-base --migrate-only --from=0 --to=1 --allow-dirty`

## Публикация библиотеки

1. Увеличить версию рабочего пространства и библиотеки `yarn version --major|minor|patch`.
2. Собрать библиотеку.
3. Пройти в собравшийся каталог: `cd /dist/ngx-register-base`.
4. npm publish.

## Настройка плагинов проекта

В проекте подключены статические анализаторы кода Eslint, Stylelint и Prettier. Настройте соответствующие плагины в вашей среде разработки для использования общих конфигураций.

## Сервер разработки

Выполните `ng serve` для запуска тестового приложения test-app. Перейдите по адресу `http://localhost:4200/`.
Приложение автоматически перезагрузится при изменении любого исходного файла.
